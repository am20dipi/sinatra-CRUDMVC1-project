# The Nail Guru

Version 1.0.0 // November 2020

## Overview

The Nail Guru is a Ruby-based, full scale Sinatra application. The goals of the project are to utilize: 
1. the MVC structure 
2. RESTful Routes
3. a SQLite Database
4. ActiveRecord
5. Sessions & Registrations (login, logout, and signup)
6. CRUD actions
